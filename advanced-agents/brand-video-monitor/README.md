## Getting Started

### Prerequisites
- Node.js 18+ and npm/yarn
- Google API key for Generative AI

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   # Add your Google API key to .env
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
